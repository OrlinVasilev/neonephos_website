# Security Native Europe

Sovereignty is a commonly encountered phrase in today's world but what does it actually mean in the context of cloud native?

Find out on **May 11th, 17:30 – 21:00 UTC+2:00** at the "Sovereign done right - common misconceptions revealed" event of the SecurityNative Europe, gracefully hosted by SAP Labs in Munich:

## Agenda

    The hype is real, so is the real need. But how to implement digital sovereignty
    
    We will address the following misconception and/or unrealistic narratives:
    
    to be sovereign everyone needs their own datacenter
    open-source = sovereign
    if the entire runtime is in my jurisdiction, it is sovereign
    I use SBOMs, so my supply chain is secure
    I cannot use any SaaS providers anymore
    to paint a more differentiated picture in this important topic, we invite everyone for a hands-on workshop , followed by this panel of renowned experts.
